# Ums
CRUD for user
CRUD for session

## Usage
1.  Register user
2.  Login user
3.  Forgot password
4.  Set Password
5.  Change password
6.  Logout
7.  Update Address
8.  update profile/media for a user
9.  Create Album
10. Create Friend


## Installation
Add this line to your application's Gemfile:

```ruby
gem 'ums'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install ums
```
